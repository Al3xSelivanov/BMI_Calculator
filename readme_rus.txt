Для работы калькулятора индекса массы тела (ИМТ) требуется компьютер под управлением ОС Windows актуальной версии.
Для запуска нажмите два раза левой кнопкой мыши на исполняемый файл (bmi_calculator.exe).
Откроется окно с графическим интерфейсом программы.

Введите свой вес в килограммах и рост в сантиметрах в отведенные для этого поля.
Поля ввода должны содержать только цифры и разделители дробной части (точки).
Поля ввода не должны содержать больше пяти знаков или больше одной цифры после запятой.
Ввод не должен заканчиваться точкой или содержать больше одной точки.
Ввод не должен начинаться с нескольких нулей подряд.
Рост не должен быть равен нулю.

Нажмите на кнопку "Рассчитать", чтобы узнать ваш ИМТ.
Снизу от кнопки отобразится ваш ИМТ и интерпретация показателя в соответствии с рекомендациями ВОЗ.
Под ними отобразится диапазон значений нормальной массы тела при введенном росте.

Чтобы сменить язык, нажмите на нужный флаг в правом нижнем углу окна.
Доступны английская и русская локализация.
